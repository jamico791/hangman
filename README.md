# Hangman

A hangman game made using React and tailwind-css

## Prerequisites

- [node & npm](https://nodejs.org/en/)

## Get Started

Clone the respository:

```
$ git clone https://github.com/jamico791/hangman.git
$ cd hangman
```

Install dependencies:

```
$ npm install
```

Start the react client:

```
$ npm run start
```
